# HorizonCyclingBridge

スマートローラートレーナーとレースゲーム（Forza Motorsport等）を双方向で接続し、現実のペダリングによるゲーム内の車の運転と、ゲーム内の地形変化（斜度）に応じたローラー負荷の自動再現を両立する、.NET 8.0 ベースの Windows 用中継アプリケーション（ミドルウェア）です。

実走感にこだわった物理シミュレーションと、ゲームの自動操舵アシストを活かすためのインテリジェントなアシスト機能を搭載しており、圧倒的に没入感の高いバーチャルサイクリング／ドライブ体験を提供します。

---

## 🚀 主な機能と特徴

### 1. ２つの動作モード
*   **シミュレーションモード (Simulation Mode) [推奨]**
    ライダーと自転車の物理モデル（総重量、重力、空気抵抗、転がり抵抗、チェーン効率）に基づいて「本来自転車であれば出るはずの目標速度」を毎フレームリアルタイムに計算し、車の速度がその速度に追従するように **PID制御** でアクセル・ブレーキを細かく自動コントロールします。
*   **アーケードモード (Arcade Mode)**
    ペダリングパワー（W）を基準パワー（FTP: Functional Threshold Power）で割り、車のアクセル開度（0〜100%）にダイレクトに反映させるシンプルな直感的操作モードです。

### 2. 没入感を高めるスマートアシスト
*   **下り坂での自動滑走アシスト（オートパイロット）**
    実車同様に、下り坂に入ると足を止めても重力によって車がスルスルと自動加速します。また、Forzaの「アクセルON時のみ機能する自動操舵アシスト」が切れてコースアウトするのを完璧に防ぎます。
*   **仮想ギアダウン機能（車の限界によるペダル負荷軽減）**
    非力な車を運転している際、登り坂で車のパワーが足りずに減速した際、ペダルが異様に重くなって漕ぎ続けられなくなる現象を防ぐため、車速の低下に合わせて「自動で軽いギアに変速する」ようにスマートローラーへ送信する斜度を自動で下げるアシストを行います。
*   **平地・下り坂での完全スピンフリー（無負荷）化**
    スマートローラーの仕様による「斜度0%でも高速回転時に発生する重い空気抵抗」を完全にシャットアウトするため、送信斜度が0%以下の局面では物理抵抗レベル自体を強制的に「0 (完全フリー)」に切り替えます。これにより、下り坂で足を高速回転させた際のスカスカと軽い滑走感を実現しています。
*   **サスペンション姿勢沈み込み・加減速ノイズの相殺**
    Forzaのテレメトリ姿勢角に含まれる、車の加速・減速時のノイズや空気抵抗によるフロントの浮き（約 -0.9% 程度の定常オフセット）を数学的に相殺し、平地を走っている時はペダルが完全にフラット（0%）になるよう補正します。

## 🛠 必要環境

1.  **Windows 10 / 11 OS**（Bluetooth LE 機能が搭載されているPC）
2.  **vJoy (Virtual Joystick)**
    *   仮想ジョイスティックドライバー。インストールし、デバイス 1 が有効化されている必要があります。
    *   `vJoyInterface.dll` が本アプリの実行フォルダ、またはシステムパスに通っている必要があります。
3.  **FTMS (Fitness Machine Service) 対応のスマートローラートレーナー**
    *   Wahoo Kickr、Tacx Neo、Zwift Hub、Sarment、各種スマートバイクなどの標準BLE FTMSプロトコルに準拠したスマートローラー。
4.  **Forza Motorsport (または Fora Horizon 5/6 等のUDPテレメトリ出力に対応したレースゲーム)**

## ⚙️ セットアップ手順

### 1. vJoy のインストールと設定
1.  [vJoyの公式サイト等](https://sourceforge.net/projects/vjoy-clear/)からインストーラーをダウンロードし、インストールします。
2.  「Configure vJoy」アプリを開き、**Device 1** を以下のように設定して「Apply」を押します。
    *   **Axes**: `X`, `Y` にチェック（RX, RYにもチェックを入れておくとゲーム内アサインの検出率が上がります）
    *   **POV Hat**: `None`
    *   **Number of Buttons**: `8` 以上
3.  Forza側の「コントロール」設定にて、仮想ジョイスティックの軸をアクセルおよびブレーキにアサインします。

### 2. Forza Motorsport のテレメトリ設定
1.  ゲーム内の **「オプション」 ➔ 「HUDとゲームプレイ」** を開きます。
2.  最下部にある以下の項目を設定します。
    *   **データのアウトプット (Data Out)**: `オン (ON)`
    *   **データアウトプットのIPアドレス**: `127.0.0.1`
    *   **データアウトプットのポート**: `5000`
    *   **データアウトプットのフォーマット**: `CAR DASH`

### 3. スマートローラーの準備
1.  スマートローラーの電源を入れます。
2.  スマートフォン等、他のデバイスのアプリ（Zwiftや専用アプリ等）とスマートローラーのBluetooth接続が切れている（ペアリング待機状態である）ことを確認してください。

## 🎮 使い方と実行手順

### 1. アプリケーションの起動
1.  コマンドプロンプトや PowerShell でプロジェクトフォルダに移動します。
2.  以下のコマンドを実行してアプリケーションを起動します。
    ```bash
    dotnet run --project src/HorizonCyclingBridge/HorizonCyclingBridge.csproj
    ```

### 2. 初期設定の入力
起動すると、コンソール上に設定用のプロンプトが表示されます。

1.  **[MODE SELECTION]**
    *   動作モードを選択します。`1`（Arcade）または `2`（Simulation）。空エンターで `2`（シミュレーション）が選択されます。
2.  **[TRAINER DIFFICULTY SELECTION]**
    *   スマートローラーの負荷再現割合（0%〜100%）を入力します。空エンターで `50%` に設定されます。（ペダルが重すぎると感じる場合は `10%` や `20%` などの低い値から始めることを強く推奨します）。

### 3. 自動接続と連動開始
*   初期設定が終わると、自動的に周囲のスマートローラーのスキャンが始まります。
*   接続が成功すると、スマートローラーの物理抵抗が初期解放され、Forzaからのテレメトリ受信（ポート5000番）の待機状態に入ります。
*   **ゲーム内でレース（走行）を開始すると、双方向連動が自動でアクティブになります！**

## ⌨️ 実行中のキーボード操作

アプリケーションのコンソールウィンドウを選択した状態で、以下のキー操作を行えます。

| キー操作 | 機能 | 動作の詳細 |
| :---: | :--- | :--- |
| **`[` キー** | 負荷難易度を 10% 下げる | スマートローラーの負荷再現割合（Difficulty）を 10% 減少させます。変更は即座にスマートローラーへ送信されます（最小 0%）。 |
| **`]` キー** | 負荷難易度を 10% 上げる | 負荷再現割合を 10% 増加させます（最大 100%）。 |
| **`M` キー** | 動作モードの動的切り替え | **「シミュレーションモード」と「アーケードモード」を交互に瞬時に切り替えます。** 走行中にいつでもペダリングの追従性や手応えを切り替えることができます。 |
| **`T` キー** | アクセル動作テスト（3秒） | vJoyを介して、アクセル開度 100% を3秒間強制送信します。Forza側のキーアサインが正しく認識されているかテストするために使用します。 |
| **`B` キー** | ブレーキ動作テスト（3秒） | vJoyを介して、ブレーキ開度 100% を3秒間強制送信します。ブレーキのキーアサインテスト用です。 |
| **`Q` キー** | アプリケーションの安全な終了 | 実行中のUDPレシーバーを停止し、スマートローラーとのBLE接続（GATTセッション）を安全にクリーンアップして終了します。※Escキーでの意図しない切断を防止する設計になっています。 |
